### HTML5 LocalStorage
- 문자열 형태로 데이터 저장. 
- ajax로 통신할 경우 문자열이기 때문에 `JSON.parse`로 json 객체로 변환.
- 하나의 세션 단위로 데이터를 저장하는 `SessionStorage` 
- 만료 기간이 없는 `LocalStorage`
- key, value로 되어 있으면 크기 제한이 있다.
- `LocalStorage 예시` 내가 검색한 검색어, 구매 내역 list.
- IE 6, 7 지원하지 않고 IE 8+ 지원.
